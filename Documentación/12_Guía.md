# 📖 Guía Maestra - Mikabel POS

Bienvenido/a a la guía oficial de **MikabelPOS**. Este documento fue diseñado específicamente para los dueños del Minimarket, explicando de forma clara y sin tecnicismos cómo el sistema protege su dinero, organiza su stock y facilita el crecimiento de su negocio.

---

## 1. El Corazón del Sistema: Seguridad y Roles

MikabelPOS está blindado para que la información sensible solo sea accesible por usted (el Administrador).

### 👥 Perfil: Administrador (Dueño)

Usted tiene el control total. Solo usted puede:

- Modificar **precios de costo** y ver cuánto gana realmente por cada producto.
- Ver los **Reportes Financieros** detallados y el historial de todas las cajas cerradas.
- **Anular ventas** por errores o devoluciones.
- Crear o eliminar productos del sistema.
- Registrar nuevos empleados.

### 👤 Perfil: Empleada (Caja)

El sistema está diseñado para que la empleada se enfoque en vender rápido y bien, sin ver datos confidenciales:

- Puede cobrar, buscar productos y gestionar libreta de fiados.
- **No puede** cambiar precios ni ver cuánto le costó a usted la mercadería.
- **No puede** borrar ventas ni eliminar deudas ya registradas (necesita su autorización).

---

## 2. Paso a Paso: Operación Diaria

### 🏧 Apertura de Caja

Al iniciar el turno, la empleada debe indicar con cuánto efectivo cuenta en la caja. Esto asegura que el arqueo final sea exacto.

### 🛒 El Punto de Venta (POS)

El sistema detecta automáticamente:

- **Ofertas**: Si el cliente lleva un maple de huevos, cobra el precio de bulto en lugar de 30 unidades individuales.
- **Pagos Mixtos**: El cliente puede pagar una parte en efectivo y otra con MercadoPago/Transferencia.
- **Recargos**: El sistema puede aplicar automáticamente el % extra por pagos con tarjeta si usted así lo configura.

### 📘 La Libreta de Fiados

Reemplazamos el papel por un sistema profesional:

- Se busca al cliente por nombre.
- El sistema muestra instantáneamente cuánto debe.
- Cada peso que se anote queda registrado con fecha y hora, imposible de "perder" u "olvidar".

### 🏁 Cierre de Caja (Arqueo)

Al terminar el día, la empleada cuenta el dinero físico. El sistema compara ese número con lo que "debería haber" según las ventas.

- Si falta dinero, el sistema lo informa como un **"Faltante"**.
- Se genera un reporte final que se guarda permanentemente para su revisión.

---

## 3. Inteligencia de Negocio: Sus Reportes

En la pestaña de **Reportes**, usted podrá ver:

- **Resumen General**: Sus ventas totales por día, semana o mes.
- **Productos Estrella**: ¿Qué es lo que más se vende y qué es lo que "no se mueve"?
- **Historial de Cajas**: Revise cada turno pasado para asegurar que no hubo irregularidades.

## 4. Próxima Etapa: Oportunidades de Mejora

Hemos diseñado un camino para que MikabelPOS siga creciendo y le dé más control sobre su negocio:

### Prioridad Alta

- [✅] **🔐 Control Fino de Permisos**: Usted podrá habilitar funciones específicas a sus empleadas (ej. corregir un stock mal cargado) de forma momentánea, manteniendo la seguridad pero ganando agilidad.
- [ ] **📦 Prioridad de Reposición**: Filtros inteligentes en el inventario para ver primero lo que se está agotando y saber qué comprar sin revisar góndola por góndola.
- [ ] **📝 Control de Pérdidas**: Registrar cuándo un producto se rompe, se vence o se consume en el local, para saber exactamente por qué falta mercadería.
- [ ] **💰 Ganancia Real (Margen Neto)**: Ver no solo cuánto vendió, sino cuánto le quedó de ganancia limpia restando el costo de la mercadería.
- [ ] **🚚 Gestión de Proveedores**: Registrar cuánto dinero le sale a cada distribuidor y controlar los pagos pendientes a proveedores.
- [ ] **⚡ Análisis de Horas Pico**: Un gráfico que le muestre a qué hora se llena el local para saber cuándo necesita más ayuda o cuándo puede reponer stock tranquilo.

### Prioridad Media

- [ ] **📈 Monitor de Deudas**: Un gráfico que le avise si la deuda de los clientes está creciendo más rápido de lo que están pagando (alerta de flujo de caja).
- [ ] **📲 Cobro por WhatsApp**: Un botón para enviarle al cliente su estado de cuenta por WhatsApp con un solo click.
- [ ] **🖼️ Fotos de Productos**: Galería visual para reconocer cada artículo con una imagen, ideal para agilizar la venta y evitar confusiones con productos similares.

### Prioridad Baja

- [✅] **📢 Mensajes para el Personal**: Podrá dejar un mensaje en el tablero de las empleadas (ej: "Recuerden reponer gaseosas" o "¡Que tengan un buen día!").
- [ ] **🎖️ Club Mikabel (Puntos)**: Sistema de puntos para premiar a sus clientes más fieles con descuentos automáticos.
- [❌] **🖥️ Múltiples Cajas**: Posibilidad de tener dos o más computadoras cobrando al mismo tiempo si el negocio crece.
- [❌] **⚖️ Conexión con Balanzas**: Que el sistema lea automáticamente el ticket de la balanza de fiambrería o verdulería.
