# Alcances y Entregables: Mikabel POS

## Alcance del Proyecto

El proyecto se enfoca en el desarrollo de una Aplicación Web Progresiva (PWA) robusta para la gestión del **Minimarket Mikabel**. El sistema se denomina **Mikabel POS**.

### Funcionalidades Core (MVP)

1. **Gestión de Inventario y Catálogo**
   - Alta, baja y modificación de productos.
   - Atributos completos: Marca, Foto, Unidad de medida, Categoría y Stock.
   - Generación de códigos internos para productos sin barcode.

2. **Historial de Ventas y POS**
   - Terminal de venta rápida con soporte para escáner láser.
   - Detección automática de precios por bulto/mayorista.
   - Historial de ventas y reportes de caja.

3. **Gestión de Clientes y Deudas (Fiados)**
   - Libreta de clientes con saldos acumulados.
   - Registro de abonos parciales o totales.

4. **Navegación y UX**
   - Aplicación instalable (PWA) para uso en móviles y PC.
   - Interfaz fluida con diseño premium y modo oscuro.

## Entregables Principales

- **Código Fuente:** Repositorio principal `mikabel-pos`.
- **Base de Datos:** Configuración de Firebase Firestore y Authentication.
- **Documentación:** Guías de usuario y especificaciones técnicas.
